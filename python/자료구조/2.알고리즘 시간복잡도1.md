자료구조와 알고리즘 성능1:가상컴퓨터+가상언어+가상코드

<문제>
    - 똑같은 알고리즘을 같은언어로 구현하더라고 하드웨어 성능에 따라 다르게 나타남.
    - 다양한 크기의 입력
<해결>
    가상컴퓨터(virtual machine)에서 가상언어(pseudo language)로 가상코드(Pseudo code)로 작성하여 같은환경에서 비교 가능

가상 컴퓨터 : Turing Machine (RAM 랜덤 액세스 머신)
    RAM : CPU+Memory+기본연산
        기본 연산 (단위시간 기준)
        : 배정, 대입, 복사 연산
        : 산술연산(+,-,*,/) 
        : 비교연산(>,<,>=,<=)
        : 논리연산(And, Or, Not)
        : 비트연산(bit-and, or, not)

가상 언어 (Pseudo/virtual languages)
    - 배정,산술,비교,논리,bit논리 등의 기본연산 표현
    - if,if else, if elif 등의 비교연산 표현
    - 반복문 제공(while, for)
    - 함수 정의,호출,return

가상코드 (Pseudo Code) : 가상의 언어로 도는 코드
    Algorithm ArrayMax(A,n)
        input : n개의 정수를 갖는 배열 A
        output : A의 수 중에서 최댓값 리턴
        currentMax=A[0]
        for i=1 to n-1 do
            if currentMax<A[i]:
                currentMax=A[i]
        return currentMax
