### 1차 비밀지도 (매우중요)
**파일:** `1차비밀지도(매우중요).py`  
**패턴:** 비트 OR 연산 + `bin()` + `zfill()` + 문자열 치환

두 지도(정수 배열)를 비트 OR로 합치면 한 줄에 끝남.  
`bin(n)[2:].zfill(n)` → 이진수 변환 후 앞자리 0 채우기.  
0→공백, 1→#으로 치환. `bin()` 결과에서 `0b` prefix 제거(`[2:]`)가 핵심.

```python
for i in range(n):
    merged = arr1[i] | arr2[i]
    row = bin(merged)[2:].zfill(n).replace('0', ' ').replace('1', '#')
    answer.append(row)
```

### 크레인 인형뽑기 게임 (카카오)
**파일:** `크레인인형뽑기(카카오).py`  
**패턴:** 스택(Stack)

board 각 열 위에서부터 첫 0이 아닌 값을 뽑아 basket 스택에 쌓음.  
바구니 맨 위(`basket[-1]`)와 새로 뽑은 인형이 같으면 `pop()` + `answer += 2`.  
`break`로 해당 열에서 하나만 뽑는 것 구현.

```python
for row in range(len(board)):
    if board[row][col] != 0:
        doll = board[row][col]
        board[row][col] = 0
        if basket and basket[-1] == doll:
            basket.pop()
            answer += 2
        else:
            basket.append(doll)
        break
```

### 에라토스테네스의 체 — 소수 찾기 (무조건복습)
**파일:** `소수 찾기(무조건복습).py`  
**패턴:** 에라토스테네스의 체

`O(n log log n)`의 고효율 소수 판별. n=1,000,000 이상 문제에서 반드시 사용.  
`range(2, int(n**0.5) + 1)`까지만 돌면 됨 — 더 큰 배수는 이미 처리됨.

```python
is_prime = [True] * (n + 1)
is_prime[0] = is_prime[1] = False
for i in range(2, int(n ** 0.5) + 1):
    if is_prime[i]:
        for j in range(i * i, n + 1, i):
            is_prime[j] = False
```

### combinations — 조합 활용 (복습)
**파일:** `삼총사.py`, `소수만들기(combination).py`  
**패턴:** `itertools.combinations`

`combinations(arr, k)`: arr에서 k개를 순서 없이 뽑는 모든 경우.  
삼총사: 3개 합이 0인지 확인. 소수만들기: 3개 합이 소수인지 `is_prime()` 함수로 확인.

```python
from itertools import combinations
for comb in combinations(nums, 3):
    if is_prime(sum(comb)):
        answer += 1
```

### Counter — 완주하지 못한 선수 (무조건복습)
**파일:** `완주하지못한선수(Counter).py`  
**패턴:** `collections.Counter` 뺄셈 / 정렬 비교

Counter 뺄셈: `Counter(participant) - Counter(completion)` → 1줄 해결.  
Counter 없이 풀 때: 양쪽 정렬 후 같은 인덱스 비교, 다르면 그 자리가 정답.

```python
# Counter 사용
answer = list((Counter(participant) - Counter(completion)).keys())[0]

# Counter 미사용
participant.sort(); completion.sort()
for i in range(len(completion)):
    if participant[i] != completion[i]:
        return participant[i]
return participant[-1]
```

### 딕셔너리 인덱스 관리 — 달리기 경주
**파일:** `달리기경주.py`  
**패턴:** `{이름: 인덱스}` 역매핑 딕셔너리

players 리스트를 O(n)으로 선형 탐색하면 시간초과 → `{name: idx}` 딕셔너리로 O(1) 조회.  
자리 바꿀 때 리스트와 딕셔너리를 **동시에** 갱신해야 함.

```python
p_idx = {name: i for i, name in enumerate(players)}
for name in callings:
    curr = p_idx[name]
    players[curr-1], players[curr] = players[curr], players[curr-1]
    p_idx[players[curr]], p_idx[players[curr-1]] = curr, curr-1
```

### ord/chr + 순환 인덱스 — 시저 암호 (복습)
**파일:** `시저암호(복습).py`, `둘만의암호.py`  
**패턴:** `ord()` / `chr()` + `% 26` 순환

`ord('A')=65`, `ord('a')=97`. 알파벳 순환은 `(ord(c) - base + n) % 26 + base`.  
둘만의암호: skip 제외한 알파벳 리스트 만들고 `% len(available)` 순환.

```python
# 시저암호
chr((ord(char) - ord('A') + n) % 26 + ord('A'))

# 둘만의암호
available = [chr(i) for i in range(97, 123) if chr(i) not in skip]
new_idx = (available.index(char) + index) % len(available)
```
