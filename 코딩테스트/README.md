# 코딩테스트

> 프로그래머스 기반 알고리즘 학습  
> 신찬수 교수 알고리즘 강의 55강 병행

## 진행 현황

| 레벨 | 풀이 수 | 상태 |
|------|--------|------|
| Level 0 | 5문제 | 완료 |
| Level 1 | 64문제+ | 완료 (카카오·삼성 기출 포함) |
| Level 2 | — | 진행 예정 (6/21~) |

## 복습 필요 문제 (중요 표시)

- `비밀지도` (비트 연산)
- `다트게임` (문자열 파싱)
- `달리기경주` (해시맵 최적화)
- `신고결과받기` (Counter 활용)
